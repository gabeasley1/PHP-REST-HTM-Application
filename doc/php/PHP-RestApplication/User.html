Class User

Description
Description | Vars (details) | Methods (details)

Class for manipulating a user object.

The user object deals with the PHP server's accounts, and not the remote server's accounts. This class mostly contains getters and setters for manipulating a user object. The password object is not stored, since it is never used for anything but an authentication method.
